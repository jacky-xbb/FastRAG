# 示例查询

不知道问啥时，从这里挑。每条都对应库里某篇具体标准，能真打中检索。
问法贴近「标准号 + 表名/指标名 + 页码」，命中越准。

## 一、查某标准某表的技术指标（最常用）

- GBT 18242-2025 弹性体塑性体改性沥青防水卷材，I 型卷材的可溶物含量要求是多少？
- GBT 23457-2017 预铺防水卷材的拉力和钉杆撕裂强度要求是多少？
- GBT 35467-2017 湿铺防水卷材的不透水性怎么规定的？
- JCT 974-2005 道桥用改性沥青防水卷材，低温柔度要求是多少度？
- JT/T 536-2018 路桥用塑性体改性沥青防水卷材的耐热性指标？

## 二、试验方法 / 检验规则 / 在第几页

- 沥青防水卷材的撕裂性能（钉杆法）怎么测？在哪个标准哪一页？（→ GBT 328.18-2007）
- 高分子防水卷材接缝剥离性能的试验步骤？（→ GBT 328.21-2007）
- 防水卷材吸水性怎么测？取样和判定规则在第几页？（→ GBT 328.27-2007）
- 沥青和高分子防水卷材抗冲击性能的试验方法？（→ GBT 328.24-2007）

## 三、历史/旧版标准（当普通文档检索，不再特殊标注，见 ADR-0005）

- 氯化聚乙烯-橡胶共混防水卷材标准的指标？（→ jc 684-1997）
- jc 684-1997 的拉伸强度要求是多少？

## 四、行业专用（铁路 / 水运 / 公路）

- 铁路桥梁混凝土桥面防水层有哪些性能要求？（→ TB/T 2965-2018）
- 铁路隧道防水板的厚度和拉伸性能要求？（→ TB/T 3360.1-2023）
- 水运工程防水卷材试验规程里，低温柔性怎么测？（→ JT/TS 232-2019）
- 沥青加铺层抗裂贴的剥离强度要求？（→ JT/T 971-2015）
